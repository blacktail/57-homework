# 57 Homework UI

## Development

### Step1 - Install nvm

Make sure you have [nvm](https://github.com/nvm-sh/nvm) installed

### Step2 - Start frontend server

Run following commands

```
$ cd 57-homework
$ nvm install
$ npm install
$ npm start
```

### Step3 - Visit app from browser

## Build

Run following commands

```
$ cd 57-homework
$ nvm install
$ npm install
$ npm run build
```
